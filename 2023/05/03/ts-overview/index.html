原始型別 連結

  • number
  • string
  • boolean
  • undefined: 未定義的空值
  • null: 代表空值
  • void: 通常用在函式型別沒有回傳的狀態
  • symbol: 想成是一個獨一無二的東西

上述的七個原始型別各自獨立，也就是誰也不屬於誰。
